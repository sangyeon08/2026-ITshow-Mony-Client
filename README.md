# 2026-ITshow-Mony-Client
2026년 IT Show Mony 클라이언트 레포지토리 입니다.

> 청년 1인 가구를 위한 스마트 생활비 도우미  
> 자동 지출 분석과 AI 소비 코칭을 통해 건강한 소비 습관을 만들어주는 웹 서비스

---

## 📌 프로젝트 소개

**Mony**는 청년 1인 가구를 대상으로 한 개인 금융 관리 웹 애플리케이션입니다.

카드 및 계좌의 지출 데이터를 기반으로 소비 패턴을 분석하고, AI 코치가 대화형 피드백을 제공하여 사용자가 자신의 소비 습관을 이해하고 개선할 수 있도록 돕습니다.

복잡한 가계부 입력 없이도 자동으로 지출을 분류하고, 예산 대비 진행률과 소비 패턴을 시각적으로 제공하여 현실적인 절약 습관을 형성할 수 있습니다.

---

## 🎯 프로젝트 목표

- 자동 지출 분류를 통해 수기 입력 부담 최소화
- 월간 예산 및 목표 설정 기능 제공
- 소비 패턴 분석 및 시각화
- AI 기반 대화형 소비 코칭 제공
- 청년 1인 가구의 생활비 절약 습관 형성 지원

---

## ✨ 주요 기능

### 🏠 온보딩
- 월간 예산 목표 설정
- 월급 입력 (선택)
- 카드/계좌 연동 (더미 데이터 기반)
- 자동 카테고리 분류 결과 확인

### 📊 대시보드
- 예산 대비 지출 진행률
- 남은 금액 및 남은 일수 표시
- 주요 소비 카테고리 확인
- AI 소비 코치 피드백 카드

### 💳 지출 내역 관리
- 거래 내역 조회
- 자동 카테고리 분류
- 카테고리 수동 수정
- 고정 지출 설정

### 📈 소비 분석
- 월별 총 지출 통계
- 카테고리별 소비 비중
- 고정 지출 vs 변동 지출 분석
- 전월 대비 증감률 확인

### 🤖 AI 소비 코치
- 질문형 소비 피드백
- 절약 팁 제공
- 소비 패턴 기반 맞춤 조언

### 🎯 목표 관리
- 월간 예산 설정
- 월급 기반 가용 예산 계산
- 목표 초과 시 알림 및 코치 피드백
